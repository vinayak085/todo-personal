## Features

- Add, edit and mark tasks as complete  
- Persistent state (via browser storage or local fallback)  
- Clean, minimal UI built with React  
- Hot Module Replacement (HMR) via Vite for fast dev iteration  
- Linting support with ESLint to keep code quality high
